<!DOCTYPE html> — объявление типа документа.

<*mеta charset="UTF-8"> — это метатег, который указывает кодировку символов для HTML-страницы

<html lang="ru"> — корень страницы на русском языке.

<head> — содержит метаданные, заголовок и стили.

<body> — видимая часть страницы.

html задаёт структуру сайта: заголовки, меню, параграфы.

CSS

отвечает за внешний вид: цвета, шрифты, расположение элементов. Использование Flexbox (display: flex) позволяет легко размещать элементы в линию и управлять их позиционированием. Метки <a> создают гиперссылки. Внутри <style> прописаны стили, чтобы оформить страницу.

margin

внешний отступ. Расстояние между элементом и соседними элементами или границами.

padding

внутренний отступ. Расстояние между содержимым элемента (например, текстом) и его границей.

background

фон можно изменять, меняя его цвет или вставляя картинки.

border-radius

делает углы скруглёнными.

background-color

устанавливает цвет или изображение на задний фон.

max width

задаёт максимальную ширину элемента, и если окно браузера шире — элемент не растягивается больше этого значения.

font-family: 'Arial', sans-serif;

это CSS-свойство, которое задаёт шрифт для текста на странице.

flex: 1;

позволяет тексту и картинке находиться на 1 уровне

div

в HTML (от division — «секция, раздел») — блочный элемент, который используется для создания контейнеров на веб-странице. Он помогает структурировать контент: объединять текст, изображения, формы и другие элементы в логические блоки.
